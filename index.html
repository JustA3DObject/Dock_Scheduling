Welcome to SmartPort

This tutorial will guide you through the main features of the port management system.

You'll learn how to monitor real-time port activity, run simulations, and optimize berth assignments.

Real-time Data & Weather

Monitor live port operations with real-time vessel tracking and weather conditions.

The system automatically updates vessel positions and weather data to help you make informed decisions.

Simulation Controls

Use the day selector to navigate between different simulation dates.

Run the daily simulation to see how the algorithm optimizes berth assignments in real-time.

Port Activity & Analytics

Track all port operations through the activity log and performance analytics.

Monitor vessels awaiting berthing and review the optimal berth allotment plan.

Berth Status Dashboard

The main dashboard shows the current status of all berths in the port.

Click on any berth card to view its complete service history and current vessel assignment.
